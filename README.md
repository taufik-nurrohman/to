Converter Utility
=================

Convert to X from Y.

### Browser

~~~ html
<script src="./@taufik-nurrohman/to/index.js"></script>
<script>
console.log(toNumber('1'));
</script>
~~~

### CommonJS

~~~ js
const {toNumber} = require('@taufik-nurrohman/to');

console.log(toNumber('1'));
~~~

### ECMAScript

~~~ js
import {toNumber} from '@taufik-nurrohman/to';

console.log(toNumber('1'));
~~~
